Whitepaper of Project

Point of Sale

In this project we solved a manual store system to auto computer software. The software have desktop and web application, both applications have same Idea, Design Layout, and Features. These features contains Adding, Removing, Searching, Editing Items to Inventry. Employee Login, Registration, and Editing Employees are some functions that facilitate to Employees. Billing records also saved to centralized DB, with respect to time and contains all items with their attributes.
